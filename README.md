# Web_API_CRUD_APPLICATION
This repository contains a CRUD Application and Web API built using ASP.NET Core MVC and Entity Framework. It demonstrates basic Create, Read, Update, and Delete operations with SQL Server database connectivity using ADO.NET. The project is designed to showcase RESTful API development, database integration, and backend architecture in .NET.
